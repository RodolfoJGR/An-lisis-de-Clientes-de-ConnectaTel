# An-lisis-de-Clientes-de-ConnectaTel
limpieza y preparacoon de datos para identificar el mejor segmento 
📖 Descripción

Este proyecto consiste en un análisis exploratorio de datos (EDA) para ConnectaTel, una empresa de telecomunicaciones. El objetivo es limpiar, transformar y analizar la información de los usuarios con el fin de comprender sus patrones de uso de llamadas y mensajes, identificar problemas en la calidad de los datos y obtener información útil para la toma de decisiones.

🎯 Objetivos
Explorar la estructura de los conjuntos de datos.
Identificar valores faltantes, sentinels y datos inconsistentes.
Limpiar y transformar los datos.
Analizar el comportamiento de los usuarios.
Clasificar a los clientes según su nivel de uso.
Visualizar los resultados mediante gráficos.

📊 Análisis realizado

Durante el desarrollo del proyecto se realizaron las siguientes actividades:

Exploración inicial de los datos.
Revisión de tipos de datos.
Conversión de columnas de fecha.
Identificación de valores nulos.
Detección de valores atípicos.
Limpieza de registros inconsistentes.
Creación de nuevas variables para el análisis.
Clasificación de usuarios según su nivel de uso.
Visualización mediante histogramas y gráficos de barras.
📈 Principales hallazgos
Se identificaron registros con valores faltantes que requirieron tratamiento.
Se detectaron inconsistencias en algunas variables de fechas.
Los usuarios presentan diferentes patrones de consumo, lo que permitió clasificarlos en grupos de Bajo uso, Uso medio y Alto uso.
Las visualizaciones facilitaron la comprensión del comportamiento general de los clientes.
